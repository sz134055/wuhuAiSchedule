## 使用

在小爱同学课表中选择教务网站导入，学校 选择安徽师范大学皖江学院，适配项目选择此项目，随后点击开始导入，在打开的界面登录你的账号即可。

导入方式一需要你点开到可以看到课表的页面，而导入方式二只需要你登录成功即可使用。
