## 使用

在小爱同学课表中选择教务网站导入，学校选择 安徽师范大学皖江学院，适配项目选择此项目。

随后点击开始导入，在打开的界面登录你的账号，**并下滑点击进入教务管理**，随后可点击下方一键导入导入课表。
